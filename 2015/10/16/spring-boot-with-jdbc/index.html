Configure a DataSource

лЁјм Җ Javaмқҳ javax.sql.DataSource мқён„°нҺҳмқҙмҠӨлҠ” Database Connectionмқ„ мң„н•ң standard methodлҘј м ңкіөн•ңлӢӨ. DataSource мқён„°нҺҳмқҙмҠӨлҘј мӮ¬мҡ©н•ҳкё° мң„н•ң кө¬нҳ„мІҙлҘј м„ нғқн•ҙм•ј н•ҳлҠ”лҚ°,

spring-boot-starter-jdbc лҳҗлҠ” spring-boot-starter-data-jpa лҘј м¶”к°Җн•ҳл©ҙ Spring Bootм—җм„ңлҠ” DataSource кҙҖлҰ¬лҘј мң„н•ң кө¬нҳ„мІҙлЎңмҚЁ tomcat-jdbcмқ„ defaultлЎң м ңкіөн•ңлӢӨ.

мқҙмҷём—җ м„ нғқн• мҲҳ мһҲлҠ” мҳөм…ҳмқҖ лӢӨмқҢкіј к°ҷлӢӨ.

  • к°ҖлҠҘн•ҳлӢӨл©ҙ Tomcat JDBC Connection Poolмқ„ м„ нғқн•ҳлҠ”кІғмқ„ к°ҖмһҘ м¶”мІңн•ңлӢӨ.
  • HikariCP м—ӯмӢң к°ҖлҠҘн•ң лҢҖм•ҲмқҙлӢӨ.
  • Commons DBCP м—ӯмӢң м„ нғқк°ҖлҠҘн•ң мҳөм…ҳмқҙлӮҳ, Production м—җм„ңлҠ” м¶”мІңн•ҳм§Җ м•ҠлҠ”лӢӨ.

Spring Bootм—җм„ңмқҳ DataSource м„Өм •

мқҙлІҲм—җлҠ” м „нҶөм Ғмқё XMLмқ„ нҶөн•ң м„Өм •мқҙ м•„лӢҢ, Spring Boot м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң Java Config лҘј нҶөн•ҙ DataSourceлҘј м„Өм •н•ҳлҠ” л°©лІ•мқ„ м•Ңм•„ліҙлҸ„лЎқ н•ҳкІ лӢӨ.

Spring Bootмқҳ @EnableAutoConfiguration м–ҙл…ён…Ңмқҙм…ҳмқ„ нҶөн•ҙ DataSource мҷҖ JdbcTemplateм—җ лҢҖн•ң Beanмқ„ лі„лҸ„лЎң л“ұлЎқн•ҳм§Җ м•Ҡм•„лҸ„ лҗҳлҠ”лҚ°, к·ёл ҮлӢӨл©ҙ мҶҚм„ұк°’мқ„ м–ҙл– н•ң л°©мӢқмңјлЎң м„Өм •н• мҲҳ мһҲмқ„к№Ң?

DataSource мқҳ м„Өм •мқҖ application.properties нҢҢмқјлӮҙм—җм„ң spring.datasource.* мҷҖ к°ҷмқҖ нҢЁн„ҙмңјлЎң м„Өм •мқҙ к°ҖлҠҘн•ҳлӢӨ. application.propertiesм—җм„ң DataSource м„Өм •н•ҳлҠ” л°©мӢқмқ„ мҳҲлҘјл“Өм–ҙ ліҙкІ лӢӨ,

Examples

лҚ”мҡұ л§ҺмқҖ м„Өм •к°’мқ„ нҷ•мқён•ҳкі  мӢ¶лӢӨл©ҙ DataSourceProperties лӮҙл¶ҖлҘј мӮҙнҺҙліҙл©ҙ лҗҳл©°, Spring Bootмқҳ AutoConfigure мқ„ нҶөн•ҙ spring.datasource.* м—җ лҢҖн•ң м„Өм •мқҙ DataSourcePropertiesм—җ м „лӢ¬лҗҳл©° мқҙлҘј нҶөн•ҙ DataSourceмқҳ л””нҸҙнҠё кө¬нҳ„мІҙмқё org.apache.tomcat.jdbc.pool.DataSource нҒҙлһҳмҠӨм—җ мһҗлҸҷмңјлЎң м„Өм •лҗң кІғмқ„ нҷ•мқён•  мҲҳк°Җ мһҲлӢӨ.

@EnableAutoConfiguration м–ҙл…ён…Ңмқҙм…ҳмқҖ @SpringBootApplicationлҘј кө¬м„ұн•ҳлҠ” мҡ”мҶҢмӨ‘ н•ҳлӮҳмқёлҚ°, @SpringBootApplicationлҠ” лӢӨмқҢкіј к°ҷмқҖ м–ҙл…ён…Ңмқҙм…ҳмңјлЎң кө¬м„ұлҗҳм–ҙ мһҲлӢӨ.

  • @Configuration : Indicates that a class declares one or more @Bean methods and may be processed by the Spring container to generate bean definitions and service requests for those
  • @EnableAutoConfiguration : Enable auto-configuration of the Spring Application Context
  • @ComponentScan : Provides support parallel with Spring XMLвҖҷs element

DataSourcePropertiesлҠ” Spring Bootм—җм„ң autoconfigureмқҙ к°ҖлҠҘн•ң мҶҚм„ұмңјлЎң лі„лҸ„мқҳ DataSourceлҘј кө¬нҳ„н•ҳм—¬ Beanмқ„ л“ұлЎқн–ҲлӢӨл©ҙ, application.properties лӮҙмқҳ spring.datasource.* мҶҚм„ұмқҖ м Ғмҡ©лҗҳм§Җ м•ҠлҠ”лӢӨ.
